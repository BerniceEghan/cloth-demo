This is an image classification demo powered by EfficientNetB0 trained on a custom clothing dataset.

## Features
- Upload any clothing image
- Model predicts 1 of 14 clothing types
- Uses TensorFlow 2.19 and Streamlit
- Fully supported on HuggingFace Spaces

## How to Use
1. Click **Upload Image**
2. Select any clothing image
3. Model outputs:
   - predicted class
   - confidence score

## Classes

- top, t-shirt, sweatshirt  
- sweater  
- cardigan  
- jacket  
- vest  
- pants  
- shorts  
- skirt  
- coat  
- dress  
- jumpsuit  
- cape  
- Not Clothing
